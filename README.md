# Smart Expense Tracker 💰

A full-stack Python web application to track daily expenses with category-wise visualization.

## 🚀 Features
- Add and delete expenses
- Category-wise expense tracking
- Automatic total calculation
- Interactive chart for expense visualization
- Show/Hide chart option
- Clean and responsive UI

## 🛠 Tech Stack
- **Backend:** Python, Flask
- **Frontend:** HTML, CSS
- **Database:** SQLite
- **ORM:** SQLAlchemy
- **Visualization:** Chart.js
